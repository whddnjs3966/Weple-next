# 새 컴퓨터 설정 가이드

## 1. 파일 복원 위치

| 백업 파일명 | 복원 경로 |
|---|---|
| `.env.local` | 프로젝트 루트 (`02-next-weple/.env.local`) |
| `claude-settings.local.json` | `02-next-weple/.claude/settings.local.json` |

## 2. .env.local 포함 키 목록

- `NEXT_PUBLIC_SUPABASE_URL` — Supabase 프로젝트 URL
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` — Supabase 익명 키
- `SUPABASE_SERVICE_ROLE_KEY` — Supabase 서비스 역할 키
- `NEXT_PUBLIC_NAVER_CLIENT_ID` / `NAVER_CLIENT_SECRET` — 네이버 로그인 및 검색 API
- `NEXTAUTH_SECRET` / `NEXTAUTH_URL` — NextAuth 설정
- `ANTHROPIC_API_KEY` — Claude AI API 키
- `GOOGLE_GENERATIVE_AI_API_KEY` — Google AI API 키

## 3. 새 컴퓨터에서 실행할 명령어

```bash
# 1) 저장소 클론
git clone <repo-url> 02-next-weple
cd 02-next-weple

# 2) 백업 파일 복원
cp <백업경로>/.env.local .
mkdir -p .claude
cp <백업경로>/claude-settings.local.json .claude/settings.local.json

# 3) 의존성 설치
npm install

# 4) 개발 서버 실행
npm run dev
```
